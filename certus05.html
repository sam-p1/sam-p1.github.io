CSS (Cascading Style Sheets) es un lenguaje de diseño gráfico que se utiliza para definir la presentación visual de documentos HTML y XML. Permite separar el contenido de la apariencia, facilitando la gestión del estilo de las páginas web. Con CSS, se puede controlar elementos como colores, tipografías, márgenes, espaciados y más, para dar un aspecto visual a la página.
